Entry Task

Task: Develop a minimal web-based sandbox toolkit for creating and testing marine AR modules.

Requirements:
  • Design a simple UI for adding, configuring, and visualizing marine AR objects (e.g., fish, coral) in a sandbox environment.
  • Implement at least one interactive feature (e.g., drag-and-drop, property editing, or live preview).
  • Document your code and approach.
  • Provide a short video demo and a brief write-up explaining your design and how it could be extended for more complex modules or features.
Evaluation: UI/UX quality, interactivity, code clarity, and documentation.
