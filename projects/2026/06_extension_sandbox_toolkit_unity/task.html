Entry Task — Extension of Sandbox Toolkit for simplifying the Development of Marine based AR Modules

Last Update: March 10, 2026

Goal

The applicants are required to create a module based in AR wherein a user can:

  • Make a selection between different shapes (cube, sphere, and cylinder)
  • Assign a behaviour to the selected shape (spin, and move)
  • Play the behavior of action items by spawning in AR

Where to start

Repository: https://github.com/Catrobat/mARine
Starting Point: Create Mode

Submission Format

This is a standalone task and doesn’t require any action on the project repo. The project repo should be used only for UX reference.

The candidates are required to send:

  • Screen capture video
  • Readme of the implementation
  • GitHub link to the project also consisting of the items given above

Evaluation Metrics

We encourage and welcome applicants who have experiences working with multiple technologies and disciplines.

The selection is primarily based on:

  • Working features in the task
  • Code quality (names, structure, readability)
  • Presentation quality (flow, explanations)
