Entry Task — Skeleton-Based Procedural Animation

Goal

Assess familiarity with basic skeletal motion concepts via a small, reproducible demo.

Task

Create a minimal demo (1–2 files) that shows skeletal-driven motion for a simple 2D/3D shape or a short animation GIF. Include a short README describing the approach (1–2 paragraphs) and how to run or view it.

Deliverables

  • GitHub repo link with demo files and README
  • One screenshot or GIF

Evaluation

Clarity, reproducibility, and correct basic concepts — not a full production system.
