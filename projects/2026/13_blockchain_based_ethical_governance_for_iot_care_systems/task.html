Entry Task — Blockchain-Based Ethical Governance

Goal

Assess basic familiarity with smart contracts, consent modeling, and simple off-chain verification.

Task (small & focused)

  1. Implement a minimal Solidity smart contract that allows: registering consent (grant), revoking consent, and querying the current consent state for an identifier.
  2. Create a short test or script that demonstrates registering and revoking consent (Hardhat tests, Truffle, or Remix + simple script).
  3. Provide a README with setup instructions and a short explanation (2–4 paragraphs) of how this contract could be used as an audit layer in an IoT care system.

Deliverables

  • GitHub repo link with the contract, tests, and README
  • Optional: a short 1–2 minute demo video or GIF showing the test run

Evaluation

We evaluate correctness, clarity of code, and thoughtfulness in the README. This task is intentionally small — we want to see awareness and correct design choices, not a production system.
