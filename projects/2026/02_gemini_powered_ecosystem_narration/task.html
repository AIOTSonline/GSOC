Entry Task — Gemini-Powered Ecosystem Narration

Goal

Quickly assess prompt & summarization skills and ability to produce a reproducible output.

Task

Using a small sample dataset (CSV/JSON with simple events), write a script that produces a 2–4 sentence summary. Include the prompt you would use (or mock) and a README describing how to run the script.

Deliverables

  • Repo with script, sample data, README, and sample outputs

Evaluation

We value clarity and reproducibility — mock outputs are acceptable if API keys are not available.
