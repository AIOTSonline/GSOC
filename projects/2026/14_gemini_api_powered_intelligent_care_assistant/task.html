Entry Task — Gemini-API–Powered Intelligent Care Assistant

Goal

Assess basic skills in structured prompt design, data ingestion, simple summarization, and role-aware filtering.

Task (small & focused)

  1. Create a small script that ingests a CSV of simple IoT activity events (timestamp + event type) and produces a short natural-language summary for one-day activity (2–5 sentences).
  2. Show one example of role-aware filtering (e.g., produce a summary for a "caregiver" vs a "relative" that differs in detail level).
  3. Include the prompt(s) you used (or a mocked Gemini call) and a README explaining how to run the script.

Deliverables

  • GitHub repo link with the script and README
  • Sample input CSV and sample outputs

Evaluation

We evaluate clarity, reproducibility, and thoughtfulness about role-aware and ethical constraints. This task is intentionally small to check for awareness and design sense.
